# Apple Reminder Copy

### About

Reconstruction (at least a try) of part of the official "Reminders" application from Apple.

- Persistant storage using CoreData
- Design similar to the official app with DarkMode support

### Technologies

- SwiftUI
- CoreData
- Combine
